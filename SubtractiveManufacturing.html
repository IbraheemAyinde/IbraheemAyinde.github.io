Project 05: Subtractive Manufacturing

In this CNC training, I was able to learn about what parts make up the Axiom machine and how to use it.it uses a software called VCarve. This Axiom also comes with a remote controller which is there to change any of the position of the spindle. Before cutting any item, you must be able to place the material of wood that you would like to use on the bed of the machine. Then you would place strips of double sided tape along the back of the material and drill a nail into each corner. This makes sure that there is less room for error with the Axiom.

Below is my CNC project design:

Fusion 360 Sketch:
I created a layer of squares and then extruded them in an incremently decresing order starting from the origin. I also sketched 7 circleds horizontally from the origin and mirrored them throughout the squares using the "rectangular pattern" tool.

First Extrusion of the Pyramid Base:
Above is how the extruding process started

Final Extrusion of the Pencil Pyramid:
After extruding all of the squares, I extruded the circles from the sketch. I used the extrusion tool "cut" option here to cut the circles through the pyramid.
I selected the drawing option which converted my 3D object to an ortographic 2D top view. Then, I downloaded it as a PDF file and exported it to Adobe Illustrator.

CNC Design in Adobe Illustrator:
When the pyramid was first opened in Adobe Illustrator, I separated every layer of my 2D design to get a proper blueprint for the CNC Router.
